# eslint-formatter-linkish

An eslint formatter that looks the same as the default `stylish` formatter, but produces linked filenames that terminal emulators can interact with.


## Install
```
$ npm install --save-dev eslint-formatter-linkish
```

## Usage

Specify `linkish` as the formatter. Using the CLI that looks like:

```
$ eslint --format=linkish file.js
```
